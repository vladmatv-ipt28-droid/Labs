## Опис завдання
Створення програми для візуалізації гармоніки з накладеним шумом з можливістю зміни параметрів через інтерактивний інтерфейс.

## Функціонал
- Генерація гармоніки y(t) = A*sin(2πft + φ)
- Накладання гаусівського шуму
- Інтерактивне керування параметрами через ipywidgets:
  - Амплітуда (0.1 - 3.0)
  - Частота (0.5 - 5.0 Гц)
  - Фаза (0 - 2π)
  - Середнє шуму (-1.0 - 1.0)
  - Дисперсія шуму (0.0 - 1.0)
  - Чекбокс для ввімкнення/вимкнення шуму
- Відображення графіка сигналу з шумом та чистої гармоніки
