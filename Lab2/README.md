## Опис завдання
Робота складається з двох частин:

### Частина 1: Аналіз VHI-індексу
- Завантаження VHI даних для 24 областей України з NOAA
- Data cleaning (заповнення пропусків, видалення HTML тегів)
- Переіндексація областей за українською абеткою
- Формування вибірок:
  - VHI для області за вказаний рік
  - VHI за діапазон років для вказаних областей
  - Пошук екстремумів (min, max, середнє, медіана)
- Візуалізація VHI для кількох областей

### Частина 2: Аналіз Electric Power Consumption
- Генерація тестових даних Individual Household Electric Power Consumption
- Data cleaning
- Формування вибірок:
  - Записи з потужністю > 5 кВт
  - Записи з силою струму 19-20A (пральна+холодильник > бойлер+кондиціонер)
  - Випадкова вибірка 5000 записів
  - Вечірнє споживання > 6 кВт після 18:00
  - Вибірка кожного 3-го з першої половини та кожного 4-го з другої
- Нормалізація та стандартизація
- Обчислення коефіцієнтів Пірсона та Спірмена
- One-Hot Encoding категоріального атрибута (період дня)
